# Secrets Setup

Required GitHub secrets:
- `CLASPRC_JSON`
- `SCRIPT_ID`
- `DEPLOY_ID`

Thiết lập bằng CLI:
```bash
gh secret set CLASPRC_JSON -R NOVATRIA/Agency-CRM-DataMigration
gh secret set SCRIPT_ID -R NOVATRIA/Agency-CRM-DataMigration
gh secret set DEPLOY_ID -R NOVATRIA/Agency-CRM-DataMigration
```
